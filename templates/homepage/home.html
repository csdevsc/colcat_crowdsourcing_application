Task Demo

Click the button below to start a sample Foci task. The demo includes the features listed below.

1 - Mechanical Turk Template

The demo will start at a page designed to simulate the Amazon Mechanical Turk environment. This includes a link to the actual task on our website, a box to input the completion code, and a submit button. This tab should remain open when you start the task, so you can come back and enter your unique key.

2 - Prescreen

The prescreen task consists of 10 questions to test if the participant is actively looking at the content of the task and has basic understanding. The participant will not be able to continue to the practice/tasks until they correctly answer all questions. If they are unable to correctly answer all questions, they will be forced to either return or abandon the HIT, which does not negatively affect their Mechanical Turk statistics.

3 - Practice

The practice task allows the participant to familiarize themself with the actual task. This consists of 3 questions that are similar to the actual task. They will be the same for each foci task.

4 - Foci Task

The Foci Task randomizes the columns presented to the user. The randomization is based on a key generated from their username. They will enter their answers for the columns. There is no validation for these columns, because some cells are blank. There is a reminder to double check their answers. We can add a popup to confirm that they want to continue to the next column. They have the option of switching between the columns.

5 - Task Completion/Submission

The task completion page will contain a unique key that needs to be entered into the initial Mechanical Turk Template page. You need to change tabs to the initial page to enter your unique completion code.

6 - Backend Demo

After entering the completion code, you can review the data that you entered. This is for demonstrative purposes only. Only admins with login details to our website will be able to access this data once we go live.
